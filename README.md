# BoazTimer Web App — No Install

This version is a single-file web app.
<details>

## Run option A — simplest

Double-click:

```txt
index.html
```

It opens in your browser and works without npm, Electron, Vite, or installation.

## Run option B — local web server

From this folder:

```powershell
python -m http.server 5173
```

Then open:

```txt
http://localhost:5173
```

## Upload / host it

You can upload `index.html` to any static hosting service:

- GitHub Pages
- Netlify
- Vercel
- any normal web server
  
  
</details>
